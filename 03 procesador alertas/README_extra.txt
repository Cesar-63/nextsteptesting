#setup inicial

1. comprobar que existe el archivo vacío "__init__.py" en la carpeta lib

2. comprobar que existe la carpeta "salida"

3. comprobar librerias indicadas en el README original

4. comprobar directorio del terminal que ejecuta el código
	debe ser específicamente esta carpeta ("03 procesador alertas")

5. comprobar que existe el archivo "alert.csv" en esta carpeta y que si se 
	modifica no arroje en ningun error

6. EN CASO DE que no funcione el archivo "alert.csv", ya sea por error de lenguaje
   o similares, recuperar uno nuevo desde la carpeta:
	"05 experimento -> DEF CON 22 -> 02 alertas Snort -> ppp -> alert"

   El archivo original debería leerse sin problemas y tomaría unas 3 horas en que
   se recorra entero
